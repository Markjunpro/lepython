## 使用__slots__

method | function
---|---
MethodType|给类绑定一个方法，s.set_age=MethodType(set_age,s)
**Notice:**
from types import MethodType,使用前要导入MethodType

class Student(object):
    __slots__=('name','age')
    
s=Student()
